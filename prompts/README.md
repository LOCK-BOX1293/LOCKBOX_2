# Prompt Pack Overview

This directory contains reusable prompts for specialist agents.

## Files

- `base_system_contract.md` — shared grounding and citation rules
- `expert_backend.md` — backend specialist behavior
- `expert_frontend.md` — frontend specialist behavior
- `expert_security.md` — security specialist behavior
- `expert_architect.md` — systems architecture specialist
- `expert_debugger.md` — incident/debug specialist

All expert prompts should be combined with `base_system_contract.md`.
